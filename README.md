"# ChatCPP" 
Админ-панель для чата на QT.
Для запуска "админки" необходимо использовать логин root или admin, - появятся новые Tab-вкладки:
- UserPrivateChat и Additional.
В первой доступны две панели, в первой сообщения для пользователя, во второй сообщения пользователю, 
пользователя можно выбрать при помощи кнопки сверху.
На вкладке Additional доступны:
- панель с выводом всех приватных сообщений чата;
- кнопка для бана пользователя.
